🥗 HealthMatrix – Smart Diet Management System

HealthMatrix is a web-based diet and nutrition management system designed to help users maintain a healthy lifestyle through structured meal planning, intake tracking, and expert guidance. It connects users with dietitians while providing tools to monitor daily nutrition and progress.


## 🚀 Features

### 👤 User Module
- User registration & login
- Daily food and calorie tracking
- Water intake monitoring
- View personalized diet plans
- Track health progress

### 🧑‍⚕️ Dietitian Module
- Manage assigned users
- Create and update diet plans
- Monitor user progress
- Provide recommendations

### 🛠️ Admin Module
- Manage users and dietitians
- System monitoring and control
- Data management


## 🧰 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  


## 📁 Project Structure

├── admin/ # Admin dashboard
├── user/ # User panel and tracking system
├── dietitian/ # Dietitian panel
├── auth/ # Authentication system (login/register)
├── config/ # Database configuration
├── sql/ # Database schema and data
├── assets/ # Static files (CSS, JS, images)
├── index.php # Entry point
